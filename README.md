# 深度分叉法（Deep Fork Thinking）

结合「分叉法」与「三大思维工具（第一性原理/苏格拉底式追问/奥卡姆剃刀）」的决策框架。

## 核心理念

**以终为始找分叉，追问三层挖本质，剃刀出鞘做减法，树状输出可落地。**

```
澄清阶段（确认制）                    分析阶段（推理制）
┌─────────────────────┐               ┌─────────────────────┐
│ Step 1  收集四要素   │               │ Step 3a 追问分叉点  │
│       ↓             │               │       ↓             │
│ Step 2  确认目标因素 │               │ Step 3b 挖本质      │
│       ↓             │               │       ↓             │
│ Step 3  识别分叉点   │               │ Step 4  奥卡姆剃刀   │
└─────────────────────┘               │       ↓             │
                                        │ 输出决策树          │
                                        └─────────────────────┘
```

## 何时使用

| ✅ 适用 | ❌ 不适用 |
|----------|----------|
| 问题杂乱无头绪 | 简单二选一 |
| 需要帮助他人理清思路 | 纯情感宣泄（只需倾听） |
| 多因素相互缠绕 | 已知答案的事实问题 |
| 需输出可落地方案 | |

## 复杂度分级

| 等级 | 分叉点 | 相关方 | 周期 | 追问层数 | 输出 |
|------|--------|--------|------|----------|------|
| 🟢 普通 | 1~2 个 | ≤2 人 | ≤7 天 | 1~2 层 | 简化 |
| 🔵 中复杂 | 2~3 个 | 3~5 人 | 1~4 周 | 2~3 层 | 标准 |
| 🔴 高复杂 | ≥4 个 | ≥6 人 | ≥1 月 | 3~5 层 | 完整 + ECPBMR |

## 核心流程

### 澄清阶段（Step 1~3）

1. **Step 1：收集四要素** — 目标/现状/约束/已尝试
2. **Step 2：确认目标与因素** — 合并目标确认与因素识别
3. **Step 3：识别分叉点** — 判断标准：取值不同则做法不同

### 分析阶段（Step 3a~4）

- **Step 3a：苏格拉底追问** — 逐轮追问，触及本质/公理则停止
- **Step 3b：第一性原理** — 抛开惯例，从零设计
- **Step 4：奥卡姆剃刀** — 精简分支，决策树输出

### 输出格式

```
最终结果：[目标]
│
├─ 分叉点1：[因素名]
│  ├─ 取值A → 本质：[本质] → 行动：[1. 2. 3.]
│  └─ 取值B → 本质：[本质] → 行动：[1. 2. 3.]
│
└─ 分叉点2：[因素名]（嵌套）
```

### 高复杂级附加：ECPBMR 框架

针对"路径断裂点"分支：

| 字母 | 含义 | 内容 |
|------|------|------|
| E | End（终局） | 量化目标 |
| P | Path（路径） | 分叉树路径 |
| B | Breakthrough（突破） | 关键节点 |
| M | Move（执行） | 谁+何时+做什么 |
| R | Review（复盘） | 7天/30天/60天检查点 |

## 三大思维工具调用

- **Step 3a 苏格拉底追问** → 挖到本质原因
- **Step 3b 第一性原理** → 剥离假设，回归基础
- **Step 4 奥卡姆剃刀** → 做减法，精简路径
